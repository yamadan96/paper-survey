## How this is organized

- `papers/YYYY/<slug>.md` — one note per paper (summary + critical reading).
- `topics/<topic>.md` — cross-paper theme pages that link out to `papers/...` with wikilinks.
- `templates/paper_template.md` — copy this when starting a new paper note.

## Conventions

- Wikilink papers with `[[papers/2024/example]]` so Quartz auto-builds the graph.
- Tag notes with `#llm`, `#vlm`, `#mlops`, `#rl`, etc. for filtering.
- Keep a `status:` frontmatter value (`reading` / `read` / `skim`) so the backlog stays visible.
